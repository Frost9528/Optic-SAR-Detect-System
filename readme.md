# 轻量化目标检测识别系统 (Lightweight Object Detection & Recognition System)
面向高分辨率光学与SAR（合成孔径雷达）卫星影像的轻量化智能目标检测与高精度几何定位系统。系统采用前后端分离架构，针对轻量化服务器与边缘计算环境进行了深度优化，可实现针对复杂背景、对抗伪装环境下的军事目标“检测、识别、切片、定位”流水线作业。

---

## 后端 (Backend)

后端基于 Python 框架开发，作为系统的核心算力层，采用管线化（Pipeline）架构处理海量大幅面遥感影像。

### 1. 预处理
预处理模块作为全流程的前置引擎，负责对多源、多模态的原始卫星影像进行质量修复与格式标准化。
* **光学影像增强**：集成自适应去噪（BM3D/双边滤波）、对比度增强（CLAHE）、直方图均衡化与拉普拉斯锐化，尽量消除气象与光照干扰。
* **SAR影像预处理**：采用专用的相干斑抑制滤波器（如 Lee/Frost 算法）进行降噪，配合动态范围调整与灰度拉伸，增强弱小雷达目标的可见性。
* **标准化处理**：实现大幅影像的自适应图像大小裁剪与全局归一化，统一输出标准格式。

### 2. 光学目标检测
面向高分辨率光学卫星影像，解决大场景、复杂背景下的目标识别问题。
* **核心算法**：采用轻量化卷积神经网络（CNN）架构，引入多尺度特征融合网络与注意力机制，极大提升对“小、暗、微”军事目标的捕捉能力。
* **主要功能**：对遥感影像目标进行快速自动检测与分类，输出目标边界框、类别及置信度。
* **结果切片**：根据检测结果的像素坐标，自动将原始大幅卫星影像精准裁切为标准的目标图像局部切片，作为后续分系统及增量识别的基础数据。

### 3. SAR目标检测
针对合成孔径雷达影像特有的散射机制，实现全天候、全天时高对抗条件下的目标辨识。
* **核心算法**：利用深度神经网络（NN）提取SAR影像的强散射点拓扑、几何形体等雷达特有特征，有效克服雷达影像的视角敏感性与几何畸变。
* **主要功能**：自动滤除由于相干斑引起的伪影虚警，对重点目标执行边界框回归与目标类别概率判别。
* **结果切片**：依据检测出的中心地理/像素坐标，快速分割并自动导出标准化的小幅目标切片，无缝对接后续的高精度目指点位保障快速生成分系统。

### 4. 高级扩展模块
* **抗伪装欺骗增量识别**：加载对抗样本防御模型，提取涂装、遮挡、伪装网下目标的隐蔽特征；引入增量学习机制（Incremental Learning），支持对新型伪装和新出现目标的快速迭代学习，而不退化原有识别能力。
* **对地自适应精确几何定位**：基于无/少控制点体制，自动对切片像对进行基于地理信息的精确匹配；动态精化有理函数模型（RFM）；结合高程数据（DEM）通过坐标变换自动生成目标区正射影像（DOM），输出军事级高精度经纬度与高程。

## 前端 (Frontend)

前端基于VUE3轻量级前端框架与 WebGIS 引擎构建，打造高复用、低延迟的图像信息智能交互终端。

### 1. 核心功能组件
* **大图高吞吐瓦片渲染**：针对大幅面 TIFF 遥感影像，采用前端切片或服务端预渲染瓦片技术，支持海量数据在多级地图上的流畅平移与无缝缩放。
* **智能多源图层管理**：采用双图层（Overlay）叠加设计，光学图层与 SAR 图层支持一键切换、对比或透明度叠加融合；目标检测框、置信度及类别标签作为矢量图层动态渲染在影像上方。
* **检测切片与目指看板**：右侧悬浮窗实时联动后端流处理结果，动态同步展示裁剪出的军事目标局部切片、地理坐标（经纬度）、识别置信度，并支持一键导出结构化研判报告。

## 数据字段与技术规范

| 数据项 | 数据格式 | 业务规则说明 |
| :--- | :--- | :--- |
| **卫星/预处理影像** | `TIFF` / `GeoTIFF` | 原始及增强影像，包含完整地理元数据、空间分辨率与轨道参数。 |
| **目标检测结果** | `JSON` | 包含目标 ID、类别 (`class`)、置信度 (`score`)、边界框坐标 `[xmin, ymin, xmax, ymax]` 及精化 RFM。 |
| **输出目标切片** | `TIFF` | 自动剪裁的局部目标图，切片文件名与对应 `JSON` 文件名保持严格一致，实现全自动化流水线匹配。 |
